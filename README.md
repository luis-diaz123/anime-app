# anime-app
Aplicación para regsitro de animes y sus estados.
En esta aplicación podras llevar un registro de los animes que gustes, con estados como "Visto", "No Visto", "En Emisión", "Abandonado", "Incompleto" y "Empezado". 
Se puede editar a tu gusto, ya sean colores y/o los registros que tenga la tabla.
De igual manera puedes agregar mas animes ya sea de forma unitaria o a gran cantidad por medio de un archivo CSV.
